Internship Projects


Prediction and Optimisation of properties of Deep Eutectic Solvent

Project Member, IIT Madras

The aim of the project is to design the most effective Deep Eutectic Solvent (DES) with optimal properties for use as a coolant in parallel flow heat exchangers. To achieve this, the project leverages advanced machine learning techniques for precise modeling. Dielectric constants are predicted using neural networks combined with a Quantitative Structure–Activity Relationship (QSAR) model that captures both 2D and 3D ion orientations. Viscosity is estimated using Random Forest algorithms. This multifaceted approach enables comprehensive optimization of DES properties, significantly enhancing performance and efficiency in heat exchanger applications and ensuring superior cooling effectiveness.

Developing Forward and Inverse Kinematics model using Quaternions

Research Intern, AMTDC, IITM Research Park

Quaternions are a powerful number system for handling 3D rotations, offering clear advantages over Euler angles and matrix transformations by avoiding gimbal lock and enhancing space complexity. This project develops software to automate robot control through inverse and forward kinematics, using quaternions for precise motion analysis. It involves creating a C++ program to process input coordinates and generate the required Euler angles for robot movement and vice versa, efficiently managing both rotation and translation with dual quaternions. The project also integrates this code into an offline framework for seamless functionality and effective robot automation.

Parallel Programming Projects


Parallelization of classic Lattice Boltzmann method

The Lattice Boltzmann method is a famous method used for solving many Computational fluid dynamics and transient heat conduction problems. This can be sped up by the use of parallelization. This paper details the use and efficiency of parallel programming in this topic. Results on OpenMP, MPI, hybrid OpenMP and MPI, CUDA(CuBLAS) and SYCL(MKL BLAS) have also been studied.

Link

Parallelization of Multigrid Method(V scheme)

The V-cycle scheme of the multigrid method is a technique for efficiently solving large-scale linear systems, often arising from PDEs in computational physics and engineering. This paper explores the implementation and optimization of parallel programming techniques applied to the V-cycle scheme using various frameworks, including OpenMP, MPI, OpenACC, CUDA, and SYCL.

Link

Parallel Cyclic Reduction of Tridiagonal systems

Parallelization of Cyclic Reduction Algorithm to solve Tridiagonal System of Linear Equation using GPU Programming Framework CUDA, OpenACC. The algorithm is divided into 3 steps, Forward elimination, Recursive Doubling and Back-substitution. The performance of the code was tested on Pade scheme in comparison with state of the art codes provided by NVIDIA.
